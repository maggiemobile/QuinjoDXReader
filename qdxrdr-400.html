  1. Handling Packages

    Printing and Saving a Package

    You may print a DX Package that you have opened the Previewer:

    Steps:

    Click on a Package to open it in the Previewer.

    Click the Print button to print the Package.

    Select your Printer for a paper copy.

    Select the Print to PDF option for a file copy.

    For the PDF option, supply a Name and destination folder on your computer.
